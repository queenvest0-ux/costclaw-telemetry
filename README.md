## 🖥️ What this app does

costclaw-telemetry helps you track OpenClaw agent costs in real time. It shows where your usage goes, how much each run costs, and what your dashboard looks like at a glance.

Use it when you want a simple way to follow LLM spend without digging through logs or spreadsheets.

## ✨ Main features

- Live cost tracking for OpenClaw agents
- Simple dashboard for usage and spend
- Clear view of token use and model cost
- Local storage with SQLite
- Fast setup on Windows
- Built for everyday use
- Easy-to-read totals and trends

## 📈 What you will see in the dashboard

The dashboard gives you a clear view of cost and usage. You can expect:

- Total spend for the day, week, or month
- Cost per run or task
- Token use by model
- Recent activity from active agents
- Trend view for rising or falling costs
- SQLite-backed local history for later review

This makes it easier to spot heavy tasks, high-cost models, and sudden jumps in spend.

## ❓ Questions you may have

### Do I need coding knowledge?
No. You can download the app, open it, and follow the on-screen setup.

### Does it work offline?
The dashboard keeps local records with SQLite. Live tracking needs the app to connect to your active OpenClaw setup.

### Can I use it for more than one agent?
Yes. You can track multiple agents if they feed data into the same dashboard setup.

### Where is my data stored?
Data stays on your Windows machine in a local SQLite database.

### What should I do if the dashboard looks empty?
Check that your OpenClaw agent is running, then confirm the app settings match your current setup.
